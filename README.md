# To_Do_Website
# 🚀 Smart To-Do & Productivity App

A feature-rich task management web application with **CRUD operations, analytics, category management, and theme customization**.

---

## ✨ Features

* ➕ Create, edit, and delete tasks (Full CRUD)
* 📋 View and manage all tasks
* ✅ Mark tasks as completed / pending
* 🗂️ Category-wise task organization
* 📊 Task analytics dashboard
* 📈 Status tracking (Completed vs Pending)
* 🌙 Theme customization (Light / Dark mode)
* 💾 Persistent storage using LocalStorage

---

## 📊 Analytics Dashboard

* Category-wise task distribution
* Completed vs Pending task statistics
* Productivity insights

---

## 🧠 Core Concepts

* CRUD Operations (Create, Read, Update, Delete)
* State management using JavaScript
* Data persistence using LocalStorage
* Dynamic DOM manipulation

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
